# Product Requirements Document (PRD)
## Snip - URL Shortener Service

### 1. 개요

#### 1.1 서비스 소개
Snip은 긴 URL을 짧게 변환하고, 클릭 통계를 제공하는 URL 단축 서비스입니다.

#### 1.2 목표
- 무료 인프라(Cloudflare)를 활용한 제로 유지비용 운영
- MVP 우선 출시 후 점진적 기능 확장
- 부가기능 출시 전까지 모든 기능 무료 제공

#### 1.3 도메인 구조
- `s.lento.dev` - 단축 URL 리다이렉트 전용
- `snip.lento.dev` - 웹 애플리케이션 (랜딩, 대시보드)

---

### 2. 사용자 스토리

#### 2.1 비회원
- 랜딩 페이지에서 서비스 소개 확인
- 가격 정책 확인
- Google 계정으로 로그인/회원가입

#### 2.2 회원 (Free)
- URL 단축 생성 (월 50개)
- 내 링크 목록 조회
- 기본 클릭 통계 확인 (최근 7일)
- QR 코드 다운로드 (기본)
- 링크 삭제

#### 2.3 회원 (Pro) - Phase 2
- URL 단축 생성 (월 1,000개)
- 커스텀 Alias 설정
- 상세 통계 (최근 90일)
- URL 만료 설정
- 벌크 생성 (100개/회)
- QR 코드 커스터마이징

#### 2.4 회원 (Business) - Phase 2
- 무제한 URL 생성
- 무제한 통계 보관
- API 접근
- 벌크 생성 (500개/회)
- QR 코드 로고 삽입

### 4. 비기능 요구사항

#### 4.1 성능
- 리다이렉트 응답 시간: < 100ms
- 페이지 로드 시간: < 2s (LCP)

#### 4.2 보안
- HTTPS 필수
- OAuth 2.0 표준 준수
- URL 유효성 검증 (악성 URL 기본 필터링)

#### 4.3 확장성
- Cloudflare 무료 한도 내 운영 (초기)
- 한도 초과 시 자연스러운 유료 전환
